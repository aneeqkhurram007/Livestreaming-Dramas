# Live Streaming Dramas

## Usage

### A live straming dramas web app made using MERN stack.

## Installation

```
git clone https://github.com/aneeqkhurram007/Livestreaming-Dramas.git
cd ./LiveStreaming-Dramas
npm install
cd ./livestreamingdramas
npm install
```

## Run

```
npm run dev
```
